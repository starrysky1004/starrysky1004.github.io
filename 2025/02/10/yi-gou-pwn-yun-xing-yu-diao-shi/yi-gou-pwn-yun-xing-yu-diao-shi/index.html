异构pwn运行与调试

pwn 异构
pwn
发布日期:   2025-02-10
文章字数:   688
阅读时长:   3 分

qemu安装

安装qemu qemu-user qemu-user-static

$sudo apt-get install qemu qemu-user qemu-user-static

安装qemu-system

$ sudo apt-get install qemu-system uml-utilities bridge-utils

运行的时候不一定要 -L 指定动态链接库

  • qemu-usr ：指定动态链接库运行

    • 安装动态链接库

      $ apt search "libc6-" | grep "powerpc"
$ sudo apt-get install libc6-powerpc-cross

  • qemu-user-static ：包含了所有必需的库和二进制文件，运行静态链接的程序

  • qemu-system ： 模拟整个计算机系统的跨平台运行

调试工具安装

sudo apt install gdb-multiarch

动态链接库冲突：set sysroot lib路径

arm

arm32

运行

qemu-arm -L /usr/arm-linux-gnueabi ./pwn

exp

context(arch='arm', os='linux', log_level='debug')
process(["qemu-arm","-g","1234","-L", "/usr/arm-linux-gnueabi", fileName])

调试

gdb filename
set architecture arm
add-symbol-file/set solib-search-path ./libc.so.6
attach pid

aarch64

环境构建

sudo apt-get install gcc-aarch64-linux-gnu g++-aarch64-linux-gnu

运行

-g指定端口，-L指定动态链接库，静态链接的程序无需该参数

qemu-aarch64 -L /usr/aarch64-linux-gnu/ ./pwn

指定 libc 文件运行：文件结构 ./libc/lib/ 下存放 ld 和 libc，-L 指定 ./libc

调试

gdb-multiarch -q ./pwn
set architecture aarch64
set endian little
add-symbol-file ./libc-2.21.so
target remote localhost:1234

exp

from pwn import *
context(arch='aarch64', os='linux', log_level='debug')
r = process(["qemu-aarch64", "-g", "1234", "-L", "/usr/aarch64-linux-gnu/", file_name])

powerpc

环境安装

安装powerpc交叉编译工具

$sudo apt install gcc-powerpc-linux-gnu binutils-powerpc-linux-gnu

安装动态链接库

$ apt search "libc6-" | grep "powerpc"
$ sudo apt-get install libc6-powerpc-cross

运行

运行powerpc 64 位是 ppc64，64位小端序是 qemu-ppc64le

$qemu-ppc -L /usr/powerpc-linux-gnu ./hello
$qemu-ppc-static ./pwn

exp

context(arch='powerpc', os='linux', log_level='debug', endian = 'big')
process(["qemu-ppc","-g","1234","-L", "/usr/powerpc-linux-gnu", file_name])

调试

$ gdb-multiarch -q -ex "set architecture powerpc:common" ./filename
set endian big / little
target remote :1234

编译

编译powerpc

$powerpc-linux-gnu-gcc ./test.c -o hello

mips

环境安装

mips 和 mips64 大端序：

$sudo apt-get install gcc-mips-linux-gnu g++-mips-linux-gnu libc6-dbg-mips-cross
$sudo apt-get install gcc-mips64-linux-gnuabi64 g++-mips64-linux-gnuabi64 libc6-dbg-mips64-cross

小端序：qemu-mipsel / qemu-mips64el

$sudo apt install libc6-mipsel-cross

运行

qemu-mipsel -L /usr/mipsel-linux-gnu ./SuperCgi

exp

context(arch='mips',os='linux',endian='little',log_level='debug')
r = process(["qemu-mipsel","-g","1234", file_name])

调试

gdb-multiarch -q ./pwn
set architecture mips
set endian little
target remote localhost:1234

编译

mipsel-linux-gnu-gcc test.c -o test

riscv

riscv64环境配置

sudo apt-get install gcc-riscv64-linux-gnu g++-riscv64-linux-gnu libc6-dbg-riscv64-cross
sudo apt install -y libc6-riscv64-cross binutils-riscv64-linux-gnu gcc-riscv64-linux-gnu

运行

在22的机器上

$qemu-riscv64 -L /usr/riscv64-linux-gnu pwn

exp

context(arch='riscv', os='linux', log_level='debug')
r = process(["qemu-riscv64", "-L", "/usr/riscv64-linux-gnu", "-g","1234", file_name])

调试

gdb-multiarch -q ./pwn
set architecture riscv
set endian little
target remote localhost:1234

r = process(['qemu-hexagon', '-L', 'libc', '-d', 'in_asm,exec,cpu,nochain', '-singlestep',
