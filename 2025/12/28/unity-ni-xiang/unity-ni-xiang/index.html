Unity逆向

前两天投的一家公司测试题就是 Unity 逆向，现学了一下做一些记录

题目信息

题目描述：一个标准的unity游戏，请使用c#等价还原 libil2cpp 中地址为 0x2434EAC 的函数。还原的代码可以正常编译。

文件类型：xapk

游戏详情：在 Google Play 找到了这款游戏，游戏规则是每次放置一堆可能存在不同颜色的六边形，当相邻堆的六边形顶部颜色相同时将翻转相同颜色的部分合并成一堆，当一堆中相同颜色的六边形达到一定数量会消除（本来还挺想玩的，结果我谷歌账号竟然玩不了？！

解题步骤

提取 apk

提取脚本如下

import os
import zipfile
import shutil

# 原始文件名：例如PA01-0018-com.nra.flyermaker-90.xapk
# 希望从中解压并提取出主apk：com.nra.flyermaker.apk
# 并将com.nra.flyermaker.apk命名为PA01-0018-com.nra.flyermaker-90.apk


# 获取当前目录下所有.xapk文件
xapk_files = [file for file in os.listdir('.') if file.endswith('.xapk')]

# 循环处理每个文件
for file in xapk_files:
    print(file)
    # PA01-0018-com.nra.flyermaker-90.xapk

    # 解压缩到当前目录下的与文件同名的文件夹中
    with zipfile.ZipFile(file, 'r') as zip_ref:
        folder_name = os.path.splitext(file)[0]
        zip_ref.extractall(folder_name)

    # 在解压缩后的文件夹中找到apk文件并进行重命名
    folder_files = os.listdir(folder_name)
    apk_files = [name for name in folder_files if name.endswith('.apk')]
    for apk_file in apk_files:

        # 找到解压后的主apk
        if os.path.splitext(apk_file)[0] in os.path.splitext(file)[0]:
            print(apk_file)  
            # com.nra.flyermaker.apk

            apk_path = os.path.join(folder_name, apk_file)
            print("old apk name : " + apk_path) 
            # old apk name : PA01-0018-com.nra.flyermaker-90/com.nra.flyermaker.apk

            new_apk_path = os.path.join(folder_name, folder_name+'.apk')
            print("new apk name : " + new_apk_path) 
            # new apk name : PA01-0018-com.nra.flyermaker-90/PA01-0018-com.nra.flyermaker-90.apk

            os.rename(apk_path, new_apk_path)

            # 将./PA01-0018-com.nra.flyermaker-90/PA01-0018-com.nra.flyermaker-90.apk 移动至 ./PA01-0018-com.nra.flyermaker-90.apk
            shutil.move(new_apk_path, "./"+folder_name+'.apk')

            # 删除./PA01-0018-com.nra.flyermaker-90/文件夹
            shutil.rmtree(folder_name)

反编译 apk

使用 Android Killer 对 config.arm64_v8a.apk 进行反编译，找到 libil2cpp.so，路径：config.arm64_v8a\Project\lib\arm64-v8a\libil2cpp.so

使用 Android Killer 对 com.gamebrain.hexasort 进行反编译，得到 global-metadata.dat，路径：com.gamebrain.hexasort\Project\assets\bin\Data\Managed\Metadata\global-metadata.dat

dump il2cpp 隐藏文件

将 global-metadata.dat 和 libil2cpp.so 放到同一个文件夹，使用 Il2CppDumper 进行 dump

运行 Il2CppDump 依次选择 libil2cpp.so 和 global-metadata.dat，得到隐藏文件，通过 script.json 和 il2cpp.h 可以恢复符号表

查看 dump.cs 搜索地址 0x2434EAC，获取到需要逆向的函数名为 Distribute

// RVA: 0x2434EAC Offset: 0x2433EAC VA: 0x2434EAC
	private void Distribute(bool reset) { }

DummyDll 中含有 Assembly-CSharp.dll

使用 dnSpy 打开 Assembly-CSharp.dll 定位到 Distribute 函数（VA = 0x2434EAC），发现函数位于 Gameplay/Cell，虽然 dnSpy 里没有具体的反编译代码，但根据含有的函数可以判断 Cell 类的作用是控制单个单元格，包括状态（锁定、解锁…）、增删合并、响应动画交互等

反编译 so

使用 IDA Pro 反编译 libil2cpp.so 。IDA 加载了半小时…

恢复符号表

IDAPro 导入 Il2CppDumper 的 ida_with_struct_py3，选择 script.json 和 il2cpp.h 即可恢复符号表

逆向分析

跳转到 0x2434EAC 看到恢复符号表的函数 Gameplay_Cell__Distribute，作用是将格子上堆叠的六边形方块按照颜色连续性进行分段分组

代码如下

using System;
using System.Collections.Generic;

namespace HexaSort.Gameplay
{
    /// <summary>
    /// 六边形方块的颜色类型枚举
    /// </summary>
    public enum HexType
    {
        None = 0,
        Red = 1,
        Blue = 2,
        Green = 3,
        Yellow = 4,
        Purple = 5,
        Orange = 6,
        Cyan = 7,
        Pink = 8
    }

    /// <summary>
    /// 六边形方块类
    /// </summary>
    public class Block
    {
        /// <summary>
        /// 方块的颜色类型
        /// </summary>
        public HexType Type { get; set; }

        /// <summary>
        /// 方块的位置索引
        /// </summary>
        public int Index { get; set; }

        /// <summary>
        /// 方块所属的格子（可为空，方块未放置时为null）
        /// </summary>
        public Cell? ParentCell { get; set; }

        public Block()
        {
            Type = HexType.None;
            ParentCell = null;
        }

        public Block(HexType type)
        {
            Type = type;
            ParentCell = null;
        }
    }

    /// <summary>
    /// 游戏格子类 - 用于存放堆叠的六边形方块
    /// </summary>
    public class Cell
    {
        /// <summary>
        /// 该格子上所有堆叠的方块列表
        /// 索引0 = 底部方块，索引n-1 = 顶部方块
        /// </summary>
        public List<Block> Blocks { get; private set; }

        /// <summary>
        /// 按颜色连续性分组后的方块列表（缓存）
        /// 索引0 = 顶部颜色组，索引n-1 = 底部颜色组
        /// </summary>
        private List<List<Block>>? _distribution;

        /// <summary>
        /// 格子在棋盘上的坐标
        /// </summary>
        public int Row { get; set; }
        public int Column { get; set; }

        public Cell()
        {
            Blocks = new List<Block>();
            _distribution = null;
        }

        /// <summary>
        /// 将格子中的方块按颜色连续性进行分组
        /// 从顶部向底部遍历，将连续相同颜色的方块归为一组
        /// </summary>
        /// <param name="reset">是否强制重新计算分组，忽略缓存</param>
        public void Distribute(bool reset)
        {
            // 判断是否需要重新计算分组：
            // 1. _distribution 为空（从未计算过）
            // 2. _distribution 中没有任何分组
            // 3. reset 参数为 true（调用者要求强制刷新）
            if (_distribution == null || _distribution.Count < 1 || reset)
            {
                // 创建新的分组列表
                _distribution = new List<List<Block>>();

                // 获取该格子上所有的方块列表
                List<Block>? blocksList = this.Blocks;

                // 如果方块列表为空，直接返回
                if (blocksList == null)
                {
                    return;
                }

                // 计算起始索引：从最后一个元素开始（即堆顶方块）
                // 注意：遍历是从顶部向底部进行的
                int currentIndex = blocksList.Count - 1;

                // 如果没有方块，直接返回
                if (currentIndex < 0)
                {
                    return;
                }

                // 记录上一个方块的颜色类型，用于判断颜色是否变化
                HexType previousColorType = HexType.None;

                // 当前正在填充的颜色分组
                List<Block>? currentColorGroup = null;

                // 从顶部向底部遍历所有方块
                do
                {
                    // 获取当前索引位置的方块
                    Block? currentBlock = blocksList[currentIndex];

                    // 安全检查：如果方块为空，终止遍历
                    if (currentBlock == null)
                    {
                        break;
                    }

                    // 获取当前方块的颜色类型
                    HexType currentColorType = currentBlock.Type;

                    // 只处理有效颜色的方块（Type != None）
                    if (currentColorType != HexType.None)
                    {
                        // 检查是否需要创建新的颜色分组
                        // 当前颜色与上一个方块的颜色不同，说明遇到了颜色边界
                        if (previousColorType != currentColorType)
                        {
                            // 创建一个新的分组列表
                            currentColorGroup = new List<Block>();

                            // 将新分组添加到 _distribution 中
                            _distribution.Add(currentColorGroup);

                            // 更新"上一个颜色"为当前颜色
                            previousColorType = currentColorType;
                        }

                        // 将当前方块添加到当前颜色分组
                        if (currentColorGroup != null)
                        {
                            currentColorGroup.Add(currentBlock);
                        }
                    }

                    // 移动到下一个方块（向底部方向）
                    currentIndex--;

                } while (currentIndex >= 0 && blocksList != null);
            }
            // 如果不需要重新计算（distribution有效且reset=false），函数直接返回，使用缓存的结果
        }

        /// <summary>
        /// 获取顶部颜色组的颜色类型
        /// </summary>
        /// <returns>顶部方块的颜色，如果没有方块则返回 HexType.None</returns>
        public HexType GetTopType()
        {
            // 确保分组已计算
            Distribute(false);

            // 检查分组是否有效
            if (_distribution == null || _distribution.Count < 1)
            {
                return HexType.None;
            }

            // 获取第一个分组（顶部颜色组）
            List<Block>? topGroup = _distribution[0];

            if (topGroup == null || topGroup.Count < 1)
            {
                return HexType.None;
            }

            // 返回顶部分组第一个方块的颜色
            return topGroup[0].Type;
        }

        /// <summary>
        /// 获取第二层颜色组的颜色类型
        /// </summary>
        /// <returns>第二层方块的颜色，如果不存在则返回 HexType.None</returns>
        public HexType GetSecondType()
        {
            // 确保分组已计算
            Distribute(false);

            // 检查是否有至少两个分组
            if (_distribution == null || _distribution.Count < 2)
            {
                return HexType.None;
            }

            // 获取第二个分组
            List<Block>? secondGroup = _distribution[1];

            if (secondGroup == null || secondGroup.Count < 1)
            {
                return HexType.None;
            }

            // 返回第二分组第一个方块的颜色
            return secondGroup[0].Type;
        }

        /// <summary>
        /// 获取所有颜色分组的类型和数量信息
        /// </summary>
        /// <returns>包含每个分组的 (颜色类型, 方块数量) 元组列表</returns>
        public List<(HexType ColorType, int Count)> GetTypes()
        {
            // 确保分组已计算
            Distribute(false);

            var result = new List<(HexType, int)>();

            if (_distribution == null)
            {
                return result;
            }

            // 遍历每个颜色分组，提取颜色和数量
            foreach (var group in _distribution)
            {
                if (group != null && group.Count > 0)
                {
                    result.Add((group[0].Type, group.Count));
                }
            }

            return result;
        }

        /// <summary>
        /// 获取顶部颜色组的方块列表
        /// </summary>
        /// <returns>顶部颜色组的方块列表，如果不存在则返回 null</returns>
        public List<Block>? GetTopGroup()
        {
            Distribute(false);

            if (_distribution == null || _distribution.Count < 1)
            {
                return null;
            }

            return _distribution[0];
        }

        /// <summary>
        /// 获取顶部颜色组的方块数量
        /// </summary>
        /// <returns>顶部颜色组的方块数量</returns>
        public int GetTopGroupCount()
        {
            var topGroup = GetTopGroup();
            return topGroup?.Count ?? 0;
        }

        /// <summary>
        /// 获取分组总数
        /// </summary>
        /// <returns>颜色分组的数量</returns>
        public int GetDistributionCount()
        {
            Distribute(false);
            return _distribution?.Count ?? 0;
        }

        /// <summary>
        /// 添加方块到格子顶部
        /// </summary>
        /// <param name="block">要添加的方块</param>
        public void AddBlock(Block block)
        {
            if (block == null)
            {
                return;
            }

            block.ParentCell = this;
            block.Index = Blocks.Count;
            Blocks.Add(block);

            // 标记分组缓存失效，下次访问时重新计算
            InvalidateDistribution();
        }

        /// <summary>
        /// 从格子顶部移除指定数量的方块
        /// </summary>
        /// <param name="count">要移除的方块数量</param>
        /// <returns>被移除的方块列表</returns>
        public List<Block> RemoveBlocksFromTop(int count)
        {
            var removedBlocks = new List<Block>();

            int actualCount = Math.Min(count, Blocks.Count);

            for (int i = 0; i < actualCount; i++)
            {
                int topIndex = Blocks.Count - 1;
                if (topIndex >= 0)
                {
                    Block block = Blocks[topIndex];
                    Blocks.RemoveAt(topIndex);
                    block.ParentCell = null;
                    removedBlocks.Add(block);
                }
            }

            // 标记分组缓存失效
            InvalidateDistribution();

            return removedBlocks;
        }

        /// <summary>
        /// 移除顶部颜色组的所有方块
        /// </summary>
        /// <returns>被移除的方块列表</returns>
        public List<Block> RemoveTopGroup()
        {
            var topGroup = GetTopGroup();

            if (topGroup == null || topGroup.Count == 0)
            {
                return new List<Block>();
            }

            return RemoveBlocksFromTop(topGroup.Count);
        }

        /// <summary>
        /// 使分组缓存失效，下次访问时将重新计算
        /// </summary>
        public void InvalidateDistribution()
        {
            _distribution = null;
        }

        /// <summary>
        /// 清空格子上的所有方块
        /// </summary>
        public void Clear()
        {
            foreach (var block in Blocks)
            {
                if (block != null)
                {
                    block.ParentCell = null;
                }
            }

            Blocks.Clear();
            InvalidateDistribution();
        }

        /// <summary>
        /// 获取格子上的方块总数
        /// </summary>
        public int BlockCount => Blocks?.Count ?? 0;

        /// <summary>
        /// 检查格子是否为空
        /// </summary>
        public bool IsEmpty => BlockCount == 0;
    }

    /// <summary>
    /// 测试程序入口
    /// </summary>
    public class Program
    {
        public static void Main(string[] args)
        {
            Console.WriteLine("=== HexaSort Distribute 函数测试 ===\n");

            // 创建测试格子
            var cell = new Cell();

            // 添加方块（从底到顶：红、红、蓝、蓝、蓝、红）
            Console.WriteLine("添加方块序列（从底到顶）：红、红、蓝、蓝、蓝、红");
            cell.AddBlock(new Block(HexType.Red));   // 索引 0（底部）
            cell.AddBlock(new Block(HexType.Red));   // 索引 1
            cell.AddBlock(new Block(HexType.Blue));  // 索引 2
            cell.AddBlock(new Block(HexType.Blue));  // 索引 3
            cell.AddBlock(new Block(HexType.Blue));  // 索引 4
            cell.AddBlock(new Block(HexType.Red));   // 索引 5（顶部）

            Console.WriteLine($"\n方块总数: {cell.BlockCount}");

            // 测试 Distribute
            Console.WriteLine("\n--- 执行 Distribute(false) ---");
            cell.Distribute(false);

            // 测试 GetTopType
            HexType topType = cell.GetTopType();
            Console.WriteLine($"顶部颜色 (GetTopType): {topType}");

            // 测试 GetSecondType
            HexType secondType = cell.GetSecondType();
            Console.WriteLine($"第二层颜色 (GetSecondType): {secondType}");

            // 测试 GetTypes
            Console.WriteLine("\n--- 所有颜色分组 (GetTypes) ---");
            var types = cell.GetTypes();
            for (int i = 0; i < types.Count; i++)
            {
                Console.WriteLine($"  分组[{i}]: 颜色={types[i].ColorType}, 数量={types[i].Count}");
            }

            // 测试 GetTopGroupCount
            Console.WriteLine($"\n顶部分组方块数: {cell.GetTopGroupCount()}");
            Console.WriteLine($"分组总数: {cell.GetDistributionCount()}");

            // 测试移除顶部分组
            Console.WriteLine("\n--- 移除顶部颜色组 ---");
            var removed = cell.RemoveTopGroup();
            Console.WriteLine($"移除了 {removed.Count} 个方块");

            // 再次查看分组
            Console.WriteLine("\n--- 移除后的分组 ---");
            types = cell.GetTypes();
            for (int i = 0; i < types.Count; i++)
            {
                Console.WriteLine($"  分组[{i}]: 颜色={types[i].ColorType}, 数量={types[i].Count}");
            }

            Console.WriteLine($"\n当前顶部颜色: {cell.GetTopType()}");
            Console.WriteLine($"剩余方块数: {cell.BlockCount}");

            Console.WriteLine("\n=== 测试完成 ===");
        }
    }
}
